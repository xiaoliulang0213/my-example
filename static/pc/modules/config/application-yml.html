application.yml配置（已过时）

app

应用自定义配置

instance：属性定义
  • id：系统id
  • code：系统代码，对应Dic_SysInfo.sysCode
  • profile：默认为main，用于区分不同环境（开发、测试、生产等）
rest接口配置
  • profile
redis配置
  • prefix：redis读写前缀，避免应用间数据冲突
错误代码配置
  • prefix：错误码前缀，错误码格式为：prefix + module + code
authority：权限配置（portal用）
  • retryTimes：登录失败超过该次数要求输入验证码
  • menuAuthority：是否控制菜单、角色权限，设置为false时用户能看到全部菜单，即仅限于开发时使用
server

spring服务配置

port：监听端口，不配置的话默认8080
logging

日志配置

level：日志级别
  • root：全局设置，支持DEBUG、INFO、WARN、ERROR等通用日志级别
  • com.xxx.xxx：某个包的设置
security
安全配置
sessions：是否创建session，portal设置为ALWAYS，其他应用设置为NEVER，共享portal创建的session
redis

redis配置

single：单节点redis配置
  • host：服务地址
  • port：服务端口
cluster：redis集群配置（同上面redis.single不能并存）
  • address：服务地址及端口列表，多个地址之间用逗号(,)分开
  • timeout：超时时间
  • maxRedirections
spring

spring配置

application

  • port：服务端口

    • maxWaitMillis
