概要说明

static-静态文本、textfield-文本域

标签资源
位于/mobile/tags/bv-static.js，mobile//tags/bv-textfield.js

公共属性定义

entity
name
clazz
attr
defaultValue
value

static特殊属性定义

text
显示内容，会覆盖value设置的值
format
show

textfield特殊属性定义

type
input类型，默认为text
validate
autocomplete
fix
输入限制，支持integer-限制只能输入数字，number-限制只能输入数字、小数点、负号，+number-限制大于0的数
blur
函数，失去焦点调用，参数为(event, value, entity)
