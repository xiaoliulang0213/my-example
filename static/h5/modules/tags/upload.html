概要说明

upload-拍照上传

标签资源
位于/mobile/tags/bv-upload.js

属性定义

entity
name
type
默认为default，可上传可拍照，可设置为camera-只能拍照（pc端无法限制，部分安卓手机无法限制）
clazz
attr
validate
url
上传url
formName
form上传文件流的名字
files
数组，初始文件数量
autoUpload
是否选择后自动上传，默认为true
uploadType
上传类型, `file`为文件上传; `base64`为以base64上传，默认为file
compress
压缩配置, `false`则不压缩，格式为：{width: 1600, height: 1600, quality: 0.8}
autoIncrease
是否自动增加图片，默认为false
allowDelete
是否允许删除图片，默认为false
delete
函数，删除图片调用
maxSize
上传文件大小限制，默认为5 * 1024 * 1024（10M）
success
函数，上传成功调用
